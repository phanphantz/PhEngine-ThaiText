convert-font.py is a python script that convert Thai font 
to adjusted font that can be used by ThaiFontAdjustor script.

You need to install python and FontForge first before using this python script.
FontForge : https://fontforge.org/en-US/downloads/

To run the script :
1) Open "FontForge interactive.console" in FontForge installed location.
2) Use command "cd" in command line to access the location of convert-font.py script.
3) Use command "ffpython convert-font.py" in command line. Then input the location and filename (with extension) of the font.
4) If the font can be converted, the adjusted font file will appear in the same location of the original file.
You can now use that file with ThaiFontAdjustor script.
